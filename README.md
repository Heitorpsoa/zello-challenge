
# Zello Challenge

Projeto do processo seletivo da empresa Zello.


## Instalação

#### Requisitos
    json-server instalado
    flutter ^2.10.5

Entre na root do diretório e execute o server
```bash
  json-server database.json --watch --port 3005
```

Em outro terminal execute o flutter

```bash
  flutter run
```
